What is programming to me?

Programming is a toolbox. And languages, frameworks and labraries are the tools in that box. A carpenter might have a hammer and nails in his toolbox, like I have HTML and CSS in my toolbox. He builds houses, I build websites and webapps. The carpenter should know which tool to use for the right job. There's no need in getting out a ten ton crane for laying the roof on a shed. Like there might not be a reason to download jQuery to write a ten line script.

What is great design?

Great design is practical, beautiful, simple and deep. Great designs purpose is first and foremost serving the user, and not the designer. If the user doesn't like it, can't figure it out og gets frustrated, the designer should make it better. Great design is ever evolving and changing - almost alive. And if the process of design ever stops, it starts deteriorating.
